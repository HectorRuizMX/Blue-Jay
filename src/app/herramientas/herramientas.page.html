Herramientas Las herramientas que se usaron para el desarrollo del Comedero fueron los siguientes:

ESP32CAM

Esta placa contiene unacamara que nos permite tomar las fotos y guardarlas en una SD

ULTRASONICO

Este sensor nos permite medir la cantidad de comida que tenemos almacenada en el comedero.

PIR

Este sensor nos permite detectar el mvimiento en el comedero para que se active la ESP32 CAM
